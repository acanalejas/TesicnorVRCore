# TesicnorVRCore

Core para las apps de VR de Tesicnor

Para la creación de personajes usar el menu de la barra principal de Unity titulado "Tesicnor"

Tenemos 4 campos en el:

	-MENU -> Para la creación de menús y pantallas de escenarios
	-Players -> Para la creación de players, tanto con mandos como con manos
	-VR UI -> Para la creación de componentes de UI como sliders, dropdowns, etc. adaptados a VR
	-EPIS -> Para la creación de las principales EPIS que solemos tener que reutilizar.
